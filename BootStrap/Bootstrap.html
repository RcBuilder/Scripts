Modals

  • Requires: bootstrap.js
  • using js: $('#myModal').modal([options]);
  • options: // data-[option] (e.g: data-backdrop="static")
    • toggle - modal (data-toggle="modal")
    • target - string - model id (e.g: data-target="#myModal")
    • backdrop
      • true - with back overlay
      • false - without back overlay
      • static - with back overlay, back click doesn't close the modal
    • keyboard - true/false - close on ESC
    • show - true/false - show on init
  • Methods:
    • modal('toggle')
    • modal('show')
    • modal('hide')
  • Events: // $('#myModal').on('[event]', [function]);
    • show.bs.modal // on show called
    • shown.bs.modal // on show completed
    • hide.bs.modal // on hide called
    • hidden.bs.modal // on hide completed
  • Modal size: modal-lg, modal-sm

Tooltips

  • Requires: bootstrap.js
  • using js: $('[data-toggle="tooltip"]').tooltip([options]);
  • pop mode:
    • MUST be initialized via script
    • use 'title' attribute as your tooltip content
  • static mode:
    • use 'tooltip' class on the wrapper element
    • use 'tooltip' role on the wrapper element
    • use 'tooltip-arrow' class for the arrow icon
    • use 'tooltip-inner' class for the content element
    • add the following rules for each static tooltip:
      • position: relative
      • display: inline-block
      • opacity: 1
  • options: // data-[option] (e.g: data-placement="left")
    • toggle - tooltip (data-toggle="tooltip")
    • placement - left, top, bottom, right or auto (e.g: data-placement="top")
    • animation - true/false - use fade transition
    • container - string - apply tooltip to specific element
    • delay - number - delay tooltip in ms
    • delay - object - delay: {'show': 500, 'hide': 100}
    • html - true/false - supports html content
    • template - string - template for the tooltip, the tooltip content will be injected into the elemnt within the template that uses the 'tooltip-inner' class
      use 'tooltip-arrow' class for the tooltip arrow
      e.g: <div class="tooltip" role="tooltip"><div class="tooltip-arrow"></div><div class="tooltip-inner"></div></div>
    • title - string - default tooltip content (if 'title' isn't present)
    • trigger - click, hover, focus or manual - the event which trigger the tooltip
      can use multiple separated by space (e.g: data-trigger="click hover")
  • Methods:
    • tooltip('show')
    • tooltip('hide')
    • tooltip('toggle')
    • tooltip('destroy')
  • Events: // $('#myTooltip').on('[event]', [function]);
    • show.bs.tooltip // on show called
    • shown.bs.tooltip // on show completed
    • hide.bs.tooltip // on hide called
    • hidden.bs.tooltip // on hide completed

Pop-Overs

  • Requires: bootstrap.js
  • using js: $('[data-toggle="popover"]').popover([options]);
  • pop mode:
    • MUST be initialized via script
  • static mode:
    • use 'popover' class on the wrapper element
    • use 'tooltip' role on the wrapper element
    • use 'arrow' class for the arrow icon
    • use 'popover-content' class for the content element
    • use 'popover-title' class for the title element
    • add the following rules for each static tooltip:
      • position: relative
      • display: inline-block
      • opacity: 1
  • options: // data-[option] (e.g: data-placement="left")
    • toggle - popover (data-toggle="popover")
    • placement - left, top, bottom, right or auto (e.g: data-placement="top")
    • animation - true/false - use fade transition
    • container - string - apply popover to specific element
    • delay - number - delay popover in ms
    • delay - object - delay: {'show': 500, 'hide': 100}
    • html - true/false - supports html content
    • template - string - template for the popover, the popover title will be injected into the element within the template that uses the 'popover-title' class
      the popover content will be injected into the element within the template that uses the 'popover-content' class
      use 'arrow' class for the popover arrow
      e.g:
      <div class="popover" role="tooltip">
         <div class="arrow"></div>
         <h3 class="popover-title"></h3>
         <div class="popover-content"></div>
      </div>
    • title - string - default popover content (if 'title' isn't present)
    • content - string - popover content
    • trigger - click, hover, focus or manual - the event which trigger the popover
      can use multiple separated by space (e.g: data-trigger="click hover")
  • Methods:
    • popover('show')
    • popover('hide')
    • popover('toggle')
    • popover('destroy')
  • Events: // $('#myPopover').on('[event]', [function]);
    • show.bs.popover // on show called
    • shown.bs.popover // on show completed
    • hide.bs.popover // on hide called
    • hidden.bs.popover // on hide completed

Accordion

  • Requires: bootstrap.js
  • using js: $('.collapse').collapse();
  • set aria-expanded on the button to be true for open collapsible contents
  • Panel Group
    • class="panel-group"
    • id="[id]"
    • role="tablist"
    • aria-multiselectable="true"
  • Panel Wrapper
    • class="panel"
  • Panel Header
    • class="panel-heading"
    • role="tab"
  • Panel Title
    • class="panel-title"
  • Panel Content
    • id="[id]"
    • class="panel-collapse collapse" // add 'in' class for open mode
    • role="tabpanel"
  • Button
    • data-toggle="collapse"
    • data-parent="#[panel group id]"
    • href="#[panel content id]"
    • aria-expanded="[true/false]" // true for the opening content
    • aria-controls="[panel content id]"
  • Classes
    • .collapse - hides content
    • .collapse.in - shows content
    • .collapsing - applied during transitions
  • options: // data-[option] (e.g: data-toggle="collapse")
    • toggle - collapse (data-toggle="collapse")
    • parent - the parent wrapper selector which uses the 'panel-group' class (e.g: data-parent="#accordion")
  • Methods:
    • collapse('show')
    • collapse('hide')
    • collapse('toggle')
  • Events: // $('#myCollapsible').on('[event]', [function]);
    • show.bs.collapse // on show called
    • shown.bs.collapse // on show completed
    • hide.bs.collapse // on hide called
    • hidden.bs.collapse // on hide completed

Carousel

  • Requires: bootstrap.js
  • using js: $('.carousel').carousel();
  • Carousel Wrapper
    • id="[id]"
    • class="carousel slide"
    • data-ride="carousel"
  • Indicators Wrapper
    • class="carousel-indicators"
  • Indicator (Item)
    • data-target="#[carousel wrapper id]"
    • data-slide-to="0" // set indicator index
    • class="active" // set active indicator
  • Slides Wrapper
    • class="carousel-inner"
    • role="listbox"
  • Slide (Item)
    • class="item"
    • use 'active' class to set item as the active slide
    • use 'carousel-caption' class to add content to the slide
  • Controls Wrapper
    • class="carousel-control"
    • use 'left' or 'right' classes to determine the control direction
    • href="#[carousel wrapper id]"
    • role="button"
    • data-slide="[prev or next]"
    • use 'glyphicon' + 'glyphicon-chevron-[left or right]' to set the arrow icon control
  • options: // data-[option] (e.g: data-interval="2000")
    • interval - number - delay time in ms
    • pause - 'hover' - pause on hover
    • wrap - true/false - set true for infinity loop
    • keyboard - true/false - set true to react to keyboard events
  • Methods:
    • carousel('cycle')
    • carousel('pause')
    • carousel(number)
    • carousel('prev')
    • carousel('next')
  • Events: // $('#myCarousel').on('[event]', [function]);
    • slide.bs.carousel // on slide changed
    • slid.bs.carousel // on slide transition completed
