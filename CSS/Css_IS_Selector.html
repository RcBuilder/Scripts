About

css is selector allows us to create a compound-selectors css rules.


Syntax

:is([selector-1], [selector-2] ... [selector-N]) ...


Browser support

Chrome: 88
Firefox: 78
Edge: 88
Safari: 14 

     
        /*
            section > h1, section > h2, section > h3, 
            article > h1, article > h2, article > h3 {
                color: red;
            }
        */
   
        :is(section, article) > :is(h1, h2, h3){
            color: red;
        }
