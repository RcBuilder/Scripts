
syntax
------
position: -webkit-sticky;
position: sticky;
top: 0;
z-index: 999;

direction
---------
top
right
bottom
left

rules
-----
the sticky element MUST be within the same wrapper of the content 

<div class="page">
	<nav class="sticky">
		...
		...
	</nav>	

	<p>
		....
	</p>
	<p>
		....
	</p>
</div>

------
