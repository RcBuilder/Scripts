transform:          
        - matrix(n,n,n,n,n,n)	
        - matrix3d(n,n,n,n,n,n,n,n,n,n,n,n,n,n,n,n)
        - translate(x,y)
        - translate3d(x,y,z)
        - translateX(x)
        - translateY(y)
        - translateZ(z)
        - scale(x,y)
        - scale3d(x,y,z)
        - scaleX(x)
        - scaleY(y)
        - scaleZ(z)
        - rotate(angle)
        - rotate3d(x,y,z,angle)
        - rotateX(angle)
        - rotateY(angle)
        - rotateZ(angle)
        - skew(x-angle,y-angle)
        - skewX(angle)
        - skewY(angle)
        - perspective(n)

        note!
        supports multiple (e.g: transform: rotate(180deg) scale(1.7);)

        ---

        transition: [property] [duration] [timing-function] [delay];
        -OR-
        transition-property
        transition-duration
        transition-timing
        transition-delay

        timing-function:
        - linear
        - ease
        - ease-in
        - ease-out
        - ease-in-out            
        - cubic-bezier(x1, y1, x2, y2)
    
   
            how to:  
            1. create css circles 
            2. set transform to move all circles on the x axis
            3. set different background-color for each circle 
            4. set transitions using different timing-function for each circle

            circle:
            <li class="bg-red linear">1</li>

            css:            
            .wrapper:hover ul.circles1 li {
                transform: translateX(400px);         
            }  

            .linear { transition: all 1.5s linear; }
            .ease { transition: all 1.5s ease; }
            .ease-in { transition: all 1.5s ease-in; }
            .ease-out { transition: all 1.5s ease-out; }
            .ease-in-out { transition: all 1.5s ease-in-out; }
            .cubic-bezier { transition: all 1.5s cubic-bezier(0, 1, 0.98, 0);  }
