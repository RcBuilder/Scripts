CSS Href value property
-----------------------

/* default */
a {
    color: #000;
}

a[href="#"] {
    color:red;
}
a[href="#a"] {
    color:green;
}
a[href="#b"] {
    color:blue;
}
a[href="https://rcb.co.il"] {
    color:purple;
}

-
