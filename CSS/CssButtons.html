            how to:  
            1. anchor
               - set as relative in order to add absolute layouts before/ after it
               - add border and color
            2. anchor (hover) 
               - change the color to fit the new layout background-color
            3. before layout
               - set empty content just to create a block 
               - set the z-index to -1 in order Not to hide the anchor text
               - stretch the bottom, right and left sides to fit the anchor size
               - hide the top side at the bottom line by setting it to 100%
            4. before layout (hover)
               - stretch the top side (a reminder: all other already stretched so now the layout is fully stretched) 
               - set the fill background-color              
               - use transition on the top property

            css:
            .design2 > a {                  
                position: relative;
                color: #000;                            
                border: solid 1px #000;                   
            }
            .design2 > a:hover {
                color: #fff;             
            }

            .design2 > a::before {                
                content: '';
                position: absolute;
                top:100%;
                bottom:0;
                left:0;
                right:0;
                z-index: -1;                        
            }
            .design2 > a:hover::before {            
                background-color: #189fda;
                top:0;
                transition: top 0.1s ease;
            }

ItemA ItemB ItemC 
   
            how to:  
            1. anchor
               - set as relative in order to add absolute layouts before/ after it
            2. before layout
               - set the content to be the link text
               - set as absolute and streach it to fit the anchor size
               - show it using opacity (opacity: 1) 
               - add transition on all properties (for the mouseOut effect)
            3. after layout
               - set the content to be the link text (on hover)
               - set as absolute and streach the top, left and bottom sides to fit the anchor size
               - pull it to the right by setting the right side to be -100% 
               - hide it using opacity (opacity: 0) 
               - add transition on all properties (for the mouseOut effect)
            4. before layout (hover)
               - pull it to the left by setting the left side to be -100% 
               - hide it using opacity (opacity: 0) 
               - add transition on all properties (for the mouseHover effect)
            5. after layout (hover)
               - pull it to the center (presented) by setting the right side to be 0
               - show it using opacity (opacity: 1) 
               - add transition on all properties (for the mouseHover effect)

            css:
            .design4 > a {                                        
                position: relative;          
            }
            .design4 > a::before {            
                content: 'before';
                position: absolute;
                top:0;            
                right:0;   
                left:0;
                bottom:0;   
                opacity:1;         
                transition: all 0.1s ease;
            }
            .design4 > a::after {            
                content: 'after';
                position: absolute;
                top:0;            
                right:-100%;   
                left:0;
                bottom:0;  
                opacity:0;          
                transition: all 0.1s ease;
            }
            .design4 > a:hover::before {            
                left:-100%;
                opacity:0;
                transition: all 0.1s ease;
            }
            .design4 > a:hover::after {            
                right:0;
                opacity:1;
                transition: all 0.1s ease;
            }
