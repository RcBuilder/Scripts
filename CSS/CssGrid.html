[Grid]
display: inline-grid;
grid-template-columns: [first-column] 100px [second-column] 300px;
grid-template-rows: [first-row] 100px [second-row] 200px;
define a layout of 2 colunms and 2 rows with aliases.
col-1 named "first-column" col-2 named "second-column" row-1 named "first-row" row-2 named "second-row"
[Grid Children]
grid-column: second-column;
grid-row: second-row;
set 1st grid child-element to be located at the second-column of the second-row
grid-column: first-column;
grid-row: second-row;
set 2nd grid child-element to be located at the first-column of the second-row
grid-column: first-column / span 2;
grid-row: first-row;
set 3rd grid child-element to take 2 columns starting from the first-column of the first-row

[Grid]
display: grid;
grid-template-columns: auto 150px;
grid-template-rows: 120px auto 60px;
grid-template-areas:
    "header header"
    "main right"
    "footer footer";
define a layout using "grid-template-areas"
after setting the grid size (rows and columns), we use areas to define the layout using names (header, footer etc.)
note: use dot (.) to define an empty area
 [Grid Children]
grid-area: header;
locate the 1st grid child-element at the "header" area grid-area: footer;
locate the 1st grid child-element at the "footer" area grid-area: main;
locate the 1st grid child-element at the "main" area grid-area: right;
locate the 1st grid child-element at the "right" area
